# 🇷🇺 Russian Language Foundation: March 4, 2026

### 1. Core Grammar (Pronouns & Time)
| Russian | Phonetic | English | Category |
| :--- | :--- | :--- | :--- |
| **Я / Ты** | Ya / Ty | I / You (sing.) | Pronoun |
| **Он / Она** | On / Ona | He / She | Pronoun |
| **Мы / Они** | My / Oni | We / They | Pronoun |
| **Вчера / Сейчас** | Vchera / Seychas | Yesterday / Now | Time |
| **Завтра** | Zavtra | Tomorrow | Time |

### 2. Verbs & Actions
| Russian | English | Examples / Tense |
| :--- | :--- | :--- |
| **Учить** | To learn | **Учу** (I learn), **Учил** (Learned) |
| **Понимать** | To understand | **Понял** (Understood) |
| **Хотеть** | To want | **Хочу** (I want), **Хотят** (They want) |
| **Любить** | To love/like | **Люблю** (I love), **Любит** (He/She loves) |
| **Использовать** | To use | Essential for Cyber/Tools |

---
### 🛠️ Grammar Note: 
- For feminine nouns ending in **-а**, change to **-у** when they are the object of a verb (e.g., *Я хочу ручку*).
- Masculine and Neuter objects usually stay the same.
