# 🇷🇺 Russian Intelligence & Cyber Lexicon

## Purpose
Specialized Russian terminology for cybersecurity, threat intelligence, and operational contexts. Not found in standard Russian textbooks.

> 💡 **How to use this:** Each term includes pronunciation, meaning, and where you'd actually see it used in the wild.

---

## 💻 Cybersecurity Operations

| Russian | Pronunciation | English | Context |
|---------|--------------|---------|---------|
| **взлом** | vzlom | hack/breach | "взлом системы" = system breach |
| **уязвимость** | uyazvimost | vulnerability | used in Russian security reports |
| **утечка данных** | utechka dannykh | data leak | common in Russian cyber news |
| **вредонос** | vredonos | malware | short slang for malicious software |
| **бэкдор** | bekdor | backdoor | borrowed from English, used in APT reports |

## 🔍 Offensive Security & Pentesting Terms

| Russian | Pronunciation | English | Context |
|---------|--------------|---------|---------|
| **эксплойт** | eksployt | exploit | "использовать эксплойт" – common in Russian pentest reports |
| **полезная нагрузка** | poleznaya nagruzka | payload | used for malware or post-exploitation code |
| **латеральное движение** | lateralnoye dvizheniye | lateral movement | moving inside a compromised network |
| **закрепление** | zakrepleniye | persistence | maintaining access after initial breach |
| **эскалация привилегий** | eskalatsiya privilegiy | privilege escalation | gaining higher access rights |
| **командный центр** | komandnyy tsentr | command and control (C2) | infrastructure used by attackers |
| **spear-phishing** | spear-fishing | spear-phishing | targeted phishing (often kept in English) |
| **watering hole** | watering hole | watering hole attack | compromising websites visited by targets |
